Deliberative Agent

  • Contains an explicitly represented, symbolic model of the world
  • Makes decisions via symbolic reasoning

Two important aspects to consider

The transduction problem

  1. That of translating the real world into an accurate, adequate symbolic description, in time for that description to be useful. (vision, speech understanding, learning, etc)

The representation/reasoning problem

  1. that of how to symbolically represent information about complex real-world entities and processes, and how to get agents to reason with this information in time for the results to be useful. (knowledge representation, automated reasoning, automatic planning, etc.)
